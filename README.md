# Kelly Finance Tracker

> ⚠️ 這是一個私人記帳練習專案（非正式產品）

## 📌 專案說明

本專案為一個 **私人小型記帳工具**，主要用途為個人日常收支紀錄與金錢管理練習。

建立此專案的目的包含：

- 測試前端 Web App 開發流程
- 練習 PWA（Progressive Web App）技術
- 建立可安裝於手機的簡易記帳工具
- 作為未來個人財務管理系統的原型（Prototype）

本專案屬於 **個人使用 / 技術練習性質**，並非正式商業系統。
